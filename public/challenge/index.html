2. How to do it

Use your current workflow.

Cut 8 clips, each 8 seconds long, one from each merged clip.
Export 8 screenshots, one for each clip.
Create 1 trailer that is 8 seconds long.
The end result should be the clips, screenshots, and a metadata list with at least path, title, and tags.
