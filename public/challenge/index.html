How it works

0. Prepare your current editing and metadata workflow.
⏱️ Start when you import the video.
1. Create 8 clips, 8 seconds each. Export each clip as watermarked and unwatermarked.
2. Create 1 trailer, 8 seconds. Export it as watermarked and unwatermarked.
3. Watermark text Contentorrr Challenge.
4. Create a metadata list with:
  • Filenames
  • Filepaths
  • SFW for all, X for watermarked, Reddit for unwatermarked
🏁 Stop when all 18 exports and the metadata list are done and in the right place.
